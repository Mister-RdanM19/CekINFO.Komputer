## Fitur Utama

Aplikasi ini mencakup lebih dari 31 poin analisis data sistem yang dibagi ke dalam beberapa kategori utama:

### 🖥️ Informasi Perangkat Keras (Core)
- **Identifikasi Mesin:** Deteksi manufaktur, model sistem, dan nomor seri motherboard.
- **Analisis Prosesor:** Informasi detail mengenai unit pemrosesan pusat (CPU), jumlah core, hingga jumlah thread aktif.
- **Detail Memori:** Informasi spesifik setiap slot memori yang terpasang (Kapasitas, Kecepatan, Manufaktur, dan Nomor Part).
- **Unit Grafis:** Deteksi kartu grafis (GPU) beserta versi driver yang terpasang.

### ⚙️ Integritas Sistem & Keamanan
- **Audit Operasi:** Informasi edisi sistem operasi, arsitektur, dan versi build secara presisi.
- **Monitoring Waktu:** Pelacakan waktu booting terakhir dan total durasi aktif (Uptime) sejak sistem dinyalakan.
- **Manajemen Pengguna:** Audit daftar akun pengguna yang terdaftar pada sistem.
- **Firmware:** Informasi vendor BIOS dan tanggal rilis firmware.

### 🌐 Jaringan & Konektivitas
- **Identifikasi Host:** Deteksi nama host dan alamat protokol internet lokal (IPv4).
- **Alamat Fisik:** Pemindaian alamat MAC untuk setiap antarmuka jaringan yang tersedia.
- **Uji Stabilitas:** Fitur pengujian latensi (Ping) untuk memverifikasi status koneksi internet.

### 📊 Performa & Pemeliharaan
- **Status Sensor:** Pemantauan penggunaan sumber daya secara real-time dan status kesehatan media penyimpanan (Disk Health).
- **Optimasi Penyimpanan:** Fitur pembersih file sementara (Temporary Files) untuk menjaga efisiensi ruang disk.
- **Audit Periferal:** Deteksi perangkat input seperti Keyboard, Mouse, Webcam, dan Monitor.

## Panduan Penggunaan

1. Gunakan file eksekusi yang tersedia pada folder distribusi.
2. Sangat disarankan untuk menjalankan aplikasi dengan hak akses **Administrator** agar sistem dapat membaca seluruh nomor seri perangkat keras dan sensor kesehatan disk secara akurat.
3. Gunakan tab navigasi untuk berpindah antar kategori informasi sistem.
4. Gunakan panel tombol aksi untuk melakukan pengujian koneksi atau pembersihan file sampah.

## Persyaratan Sistem

- **Sistem Operasi:** Windows 10 / 11 (Semua Versi)
- **Hak Akses:** Administrator (Sangat disarankan untuk fitur audit mendalam)
